# 🐍 Clicker Game (Python Tkinter)

This is a simple clicker game made with Python and Tkinter.

## 🎮 About the project

In this game, you click a button to earn points.  
When you reach a certain number of clicks, your click power increases, and each click gives more points.

## ⚙️ Features

- Simple GUI using Tkinter
- Click button to increase score
- Click power system (upgrades)
- Reset button
- Basic progression system

## 🚀 How it works

- Each click increases your score
- After reaching a milestone (for example 100 clicks), click power increases
- You can reset your progress anytime
- Upgrade system makes clicks stronger over time

2. Run the file:
